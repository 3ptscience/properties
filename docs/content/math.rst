Math Properties
===============

.. note::

    Math Properties require :code:`numpy` and :code:`vectormath` to be
    installed. This may be installed with :code:`pip install properties[full]`,
    :code:`pip install properties[math]`, or
    :code:`pip install numpy vectormath`.
